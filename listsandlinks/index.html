Creating a Webpage on the Production Server

  1. Create a new file in repository and name the file index.html
  2. Write necesary codes to create the new webpage
  3. Valinate the codes for index.html
  4. Create another file called main.css
  5. Add link element to the head of index.html
  6. Add codes to main.css to add elements and properties
  7. Validate codes for main.css
  8. Type sftp username@cmis-hsteacher in terminal to upload files using sftp
  9. Type put (filename) to upload a file
  10. View webpage created using production server by typing cmis-hsteacher/username/
